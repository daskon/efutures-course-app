## Students and Course Management app

 ### Environment 
 
 - Laravel 9 
 - Livewire
 - Jetstream
 - MySQL

 ### How to run ? 

 - ```composer install```
 - ```npm install```
 - ```npm run dev```
 - create .env file in the root directory and the file should update as below 
    - DB_DATABASE= {add the DB name}
    - DB_USERNAME=root
 - ```php artisan key:generate```
 - ```php artisan migrate```
 - ```php artisan serve```
